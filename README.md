# Aquarium Cortes Especiais — Header e Hero

Primeira versão estática e responsiva do site, contendo somente o cabeçalho e a seção principal.

## Visualização

Abra `index.html` diretamente no navegador ou publique a pasta em qualquer hospedagem estática.

## Estrutura

- `index.html`: estrutura e conteúdo
- `styles.css`: identidade visual e responsividade
- `script.js`: cabeçalho ao rolar e menu mobile
- `assets/`: logo, imagem principal e ícone do site

Os links do menu para as próximas seções já estão preparados para receber os respectivos IDs futuramente.
